Project Goals

  • Create example destinations for users to look through
  • Create attractions for each destination for users to get ideas of what to do on their vacation
  • Create a page where users can build their vacation if they cannot decide where to go

Project details

Project Description:

This is a website that will recommend destinations from around the world based on the user's preferences! Will include images, descriptions, activities, and average costs for each trip.

Purpose:

To help users find travel options that match their interests. Makes planning trips much more convenient and fast.
