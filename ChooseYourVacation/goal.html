Project Goals

  • Create example destinations for users to look through

    Our team was able to complete this goal. On our website, the explore page allows users to see 10 different destinations that users can look through. With each destination, a user can see how much a round-trip flight will be from the US.

  • Create attractions for each destination for users to get ideas of what to do on their vacation

    Our team was able to accomplish this goal. On our explore page, a user can click a destination image card. When clicked, it will show a list of 10 attractions a vacation goer can explore while traveling to a certain destination. Within each attraction, a user will be able to know how much the attraction costs.

  • Create a page where users can build their vacation if they cannot decide where to go

    On the build your vacation page, a user is able to select four parameters. Those parameters include two recreational activities, a type of landmark, and the climate that they would like to be in on vacation.

Project details

Project Description:

This is a website that will recommend destinations from around the world based on the user's preferences! Will include images, descriptions, activities, and average costs for each trip.

Purpose:

To help users find travel options that match their interests. Makes planning trips much more convenient and fast.
